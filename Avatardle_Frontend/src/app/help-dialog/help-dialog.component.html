Goal

Guess the character based on different characteristics!

Color Indicators

Incorrect - when the character traits don't match.
Partially correct - only applicable to the affiliations category, where only some affiliations may match.
Correct - when the character traits do match.

Categories

Gender

Most characters fall under male or female, other entities like spirits that don't have a gender would be denoted with none.

Nationality

This refers to where a character originates. Keep in mind that sometimes the place of origin may not be a clearly defined nation (like the spirit world) so they would be denoted with none.

Bending Element

This only includes the 4 basic bending elements: Water, Earth, Fire, and Air.

Affiliations

Affiliations refer to traits or charateristics that relate to the character. Every character has at least 1 affiliation. Regardless of how many afflliations the character has, only a maximum of 3 affiliations are shown per character. If a character has more than 3 affiliations, 3 affiliations will be chosen at random from their pool.

Note about the color indicators: 0 matching affiliations = Incorrect, 1-2 matching affiliations = Partially Correct, 3 matching affiliations = Correct

First Appearance

This refers to the name of first episode that the character appears in. In addition, there is an arrow indicator that states whether the first appearance of the target character takes place before or after the current episode.
